# 订单终止交付

此模型在不调整 order 表结构的基础上“做加法”，承载订单终止交付的过程。
OrderEnd
---------------------------------------------------------------------------

Column                      | Type  | Note
----------------------------|-------|-------
`id`                        | int   | 和 `order` 共用主键
`reason`                    | str   | 
`status`                    | int   | 1: 已创建、9: 已完成

OrderEndItem
---------------------------------------------------------------------------

Column                      | Type  | Note
----------------------------|-------|-------
`id`                        | int   | 
`order_end_id`              | int   | 
`name`                      | str   | 品名
`specification`             | str   | 规格
`quantity`                  | int   | 终止交付数量
`goods_id`                  | int   | 注意：这里没有外键约束（goods 记录可能会删除）
