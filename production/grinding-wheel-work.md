# 砂轮工单

结构
--------------------------------------------------------------------------
### Schema

Column                              | Type      | Null | Note
------------------------------------|-----------|------|-------
`id`                                | int       | No   | 
`mix_id`                            | int       | No   | 
`status`                            | int       | No   | Lookup `grinding-wheel-work-status` 
`pressed_by`                        | int       | No   | 
`process`                           | int       | No   | 

状态：已创建(1)、已生效(2)、已作废(4)

操作
--------------------------------------------------------------------------
### 退回
`refuse-mix`. 混料员在交付环节回传在选错压块员的情况，通过此操作，可以让压块员将工单退回，以便混料员重新选择。

此操作就是交付操作的逆操作，将 `pressed_by` 重置为 null.
