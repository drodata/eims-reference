# 订单附加服务

---------------------------------------------------------------------
### OrderService Schema
Column                              | Type      | Null | Note
------------------------------------|-----------|------|-------
`id`                                | int       | No   | 
`order_id`                          | int       | No   | 订单编号
`service_code`                      | string    | No   | 如 `coating`

考虑到一个订单可能存在多种服务，设计成一对多的关系。
