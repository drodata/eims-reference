# Industry 行业

--------------------------------------------------------------------------
### Schema
Column                              | Type      | Null | Note
------------------------------------|-----------|------|-------
`id`                                | int       | No   | 
`name`                              | string    | No   | 
`parent_id`                         | int       | Yes  | 上级节点

--------------------------------------------------------------------------
