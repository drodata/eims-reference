# 财务
四个主要角色：
角色/权限               | Description   
------------------------|---------------------------------
`cfo`                   | 财务主管
`cashier`               | 出纳 (负责支出)
`accountant`            | 会计 (负责收款)
`finance`               | 上面三个角色的共有子权限

* [支出单](/finance/cost.md)
